# BookCut Pro

Professional salon booking management system with online reservations, staff scheduling, and skill-based appointment matching

## Quick Start

```bash
# Install dependencies
npm run setup

# Start development servers
npm run dev
```

The frontend will be available at http://localhost:5173
The backend API will be available at http://localhost:3001

## Project Structure

```
├── frontend/          # React + Vite frontend
│   ├── src/
│   │   ├── pages/    # Page components
│   │   ├── components/ # Reusable components
│   │   └── api/      # API client
│   └── ...
├── backend/           # Express.js backend
│   ├── routes/       # API routes
│   ├── database.js   # SQLite database
│   └── server.js     # Express server
└── scripts/          # Helper scripts
```

## Features

Online booking widget with shareable link, staff management with skill-based service assignments, intelligent appointment scheduling that matches services with qualified staff, real-time availability calendar, automated booking confirmations and reminders, customer management system, service catalog with pricing, staff schedule management, booking analytics and reports, mobile-responsive design for customers, and admin dashboard for salon owners.

## Tech Stack

- **Frontend**: React 18, Vite, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: SQLite
- **Styling**: Tailwind CSS
